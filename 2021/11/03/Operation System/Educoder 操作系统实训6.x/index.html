    第三关：编写一个显示文件内容的程序

    实验方法

    编写如图所示的代码，将代码放入虚拟机中，按要求在/etc/rc中添加一行文字/usr/root/a.out（a.out处是可执行文件的名字），如果编译时指定了文件名，就是指定的可执行文件名。代码之外的其它操作见这里。

    修改得到的/etc/rc文件为

    /etc/update &
echo "/dev/hd0 /" > /etc/mtab
echo "Ok. "
/usr/root/a.out

    mycat代码为

    #include <stdio.h>
#include <string.h>
#include <fcntl.h>
#include <unistd.h>
#include <stdlib.h>
#define MSG_LEN 80
#define FILE_FROM "/usr/root/hello.c"
#define FILE_TO "/usr/root/hello-new.c"

int main() {
    char msg[MSG_LEN + 10];
    int fd, num;
    printf("\n\rHi, press ENTER to continue...\n\r");
    read(0, msg, 1);
    if ((fd = open(FILE_FROM, O_RDONLY, 0)) < 0) {
        printf("Open file error.\n\r");
        exit(0);
    }
    if ((num = read(fd, msg, MSG_LEN)) < 0) {
        printf("Read file error.\n\r");
        exit(0);
    }
    close(fd);
    msg[num] = 0;
    printf("Content of file \"%s\":\n\r---\n\r%s---\n\r", FILE_TO, msg);
    return 0;
}
